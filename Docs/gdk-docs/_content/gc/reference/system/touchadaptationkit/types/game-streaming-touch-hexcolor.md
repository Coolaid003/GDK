# HexColor

String containing hexadecimal representation of RGBA color value to be referenced.

## Properties

`value` - _string_.

## Sample

```JSON
"#0e7a0d"
```

## Remarks

Colors are specified in RGBA format.

All of the following color representations would be equivalent:

- #0099ff
- #0099ffaa
- #09f,
- #09fa
